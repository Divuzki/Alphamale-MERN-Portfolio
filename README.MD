# Full Stack Portfolio Website with HD Video Streaming capability

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Reactjs
- Tailwindcss
- Snowpack
> Media Server is Amazon Web Service Bucket
#

> ### Backend API ENPOINTS
> `GET /api/media`
<br />
> `PATCH /api/media/:id`
<br />
> `DELETE /api/media/:id`
<br />
```javascript
// POST /api/media
{
    title: String, 
    video: File,
    image: File
}
```
## Video Streaming Enpoints
> `GET /api/media/stream/:id`
<br />
