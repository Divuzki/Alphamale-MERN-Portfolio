# Alphamale-MERN-Portfolio (A Full Stack Portfolio Website with HD Video Streaming Capability)

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Reactjs
- Tailwindcss
- Snowpack
> Media Server is Amazon Web Service Bucket
#

> ### Backend API ENPOINTS
> `GET /api/media`
>
> `PATCH /api/media/:id`
>
> `DELETE /api/media/:id`
>
```javascript
// POST /api/media
{
    title: String, 
    video: File,
    image: File
}
```
## Video Streaming Enpoints
> `GET /api/media/stream/:id`
>
